# 🖥️ Innoven Support - Admin Panel

A modern, high-performance administrative dashboard built with React and Vite. This panel provides complete oversight and control over the service and installation lifecycle, enabling efficient management of customers, products, and field operations.

---

## 🏗️ Tech Stack

| Layer           | Technology                       |
| --------------- | -------------------------------- |
| **Framework**   | React.js (Vite)                  |
| **Styling**     | Tailwind CSS                     |
| **Icons**       | Lucide React                     |
| **Animations**  | Framer Motion                    |
| **State/Auth**  | Custom React Hooks & Context API |
| **HTTP Client** | Axios                            |

---

## 🚀 Key Features

### 1. 📊 Centralized Dashboard

- Real-time analytics and performance metrics.
- High-level overview of pending, assigned, and completed tasks.

### 2. 📺 Installation Management

- **Full Lifecycle Tracking:** Monitor tickets from "Pending" to "Installation Completed."
- **Visual Proof:** View photo evidence uploaded by engineers directly from the dashboard.
- **Timeline History:** Detailed event logs showing exactly when an engineer was assigned or when an OTP was verified.

### 3. 🔧 Service & Repair Module

- **Integrated Assignment:** Powerful tools to assign field engineers to repair and service visits.
- **Status Control:** Update and track the progress of complex repair tasks.
- **Proof Verification:** Verify service completion with on-site photo proofs.

### 4. 👥 Entity Management

- **Customers:** Manage customer profiles and linked user accounts.
- **Products:** Register new products and track their service history.
- **Engineers:** oversee the workforce and their active assignments.

---

## 🛠️ Module Overview

- **Installations:** Manage the deployment of new products and verify successful setup.
- **Service Requests:** Handle repair requests, maintenance visits, and customer issues.
- **Customers & Products:** Maintain a clean database of users and the equipment they own.
- **Assignment System:** Smart modal-based workflow to assign the right engineer to the right task.

## 🚀 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Configure API:**
   Ensure your backend URL is correctly set in `src/services/api.js`.
3. **Launch Development Server:**
   ```bash
   npm run dev
   ```
4. **Production Build:**
   ```bash
   npm run build
   ```
